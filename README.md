# 📊 Stock Data Dashboard & LSTM Prediction

This project is a Stock Market Analysis Dashboard built using Streamlit, featuring real-time visualization and LSTM-based price prediction.

---

🚀 Features

- 📈 Interactive stock data visualization
- 📊 Technical indicators:
  - Moving Averages (MA20, MA50)
  - RSI (Relative Strength Index)
  - MACD
- 🔮 LSTM Model for stock price prediction
- 📉 Actual vs Predicted price comparison graph

---

📸 Demo Output

- Data Preview Table
- Stock Price Charts
- LSTM Prediction Graph (Actual vs Predicted)

---

🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- TensorFlow / Keras

---

📂 Project Structure

├── app.py
├── stock_data.csv
├── requirements.txt
├── README.md

/
.git clone 
/

cd your-repo-name=Real-Stock-Market-Insight-Dashboard
/


2. Install dependencies:

pip install -r requirements.txt

3. Run the app:

streamlit run app.py

---

☁️ Deploy on Streamlit Cloud

1. Go to: https://streamlit.io/cloud
2. Connect your GitHub account
3. Select repository
4. Set:
   - Branch → "main"
   - File → "app.py"
5. Click Deploy
