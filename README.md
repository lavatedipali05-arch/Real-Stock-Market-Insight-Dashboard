# 🚀 Pro Real-Time Stock Dashboard

Live Stock Market Analysis Dashboard built using Streamlit, Plotly, and Machine Learning.

## 📌 Features

- 📈 Real-Time Stock Data
- 🕯️ Candlestick Charts
- 📊 RSI Indicator
- 📉 MACD Indicator
- 🤖 ML Price Prediction
- 💹 Buy/Sell Signals
- 🌙 Dark Theme UI
- ☁️ Streamlit Cloud Deployment

---

## 🛠️ Technologies Used

- Python
- Streamlit
- yFinance
- Plotly
- Pandas
- NumPy
- Scikit-learn
- TA Library
